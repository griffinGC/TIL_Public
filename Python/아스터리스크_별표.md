# 아스터리스크(*)

- 언팩(unpack) 의미

  - 반대로 함수의 파라미터로 올때는 패킹도 가능

- 시퀀스를 풀어헤치는 연산자

- 튜플이나 리스트를 언패킹하는데 사용

  - 1개는 튜플 또는 리스트 등의 시퀀스 언패킹
  - 2개는 키/값 페어 언패킹

- 대입할때는 남은 요소 전체를 패킹하는데 사용됨

  > https://python.bakyeono.net/chapter-5-5.html

  ```python
  >>> *a, b = [1,2,3,4] # b에 한개만 저장하고 나머지는 다 a에 저장
  >>> a
  [1, 2, 3]
  >>> b
  4
  ```
