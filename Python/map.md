# Map

### 자료구조 형태의 Map

- 자료 구조 형태의 경우, key, values를 가지고 있음
- 사용할때는 딕셔너리 형태로 사용
  - 자세한 것은 [딕셔너리](./딕셔너리.md) 파일 참조



### map 함수

- 기본형
  
  - `map(function, 반복가능한_자료형)`
  
- 자료형 안에 있는 각각의 원소가 `function`을 수행

- `map()` 함수를 이용하면 형변환을 할 수 있음

  ```python
  >>> a = [1.2, 2.5, 3.7, 4.6]
  # int 형으로 형변환
  >>> a = list(map(int, a))
  >>> a
  [1, 2, 3, 4]
  ```
