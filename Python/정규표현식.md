- `re.findall(pattern, string, flags)`
  - 문자열중 패턴과 일치하는 모든 부분을 찾음
- 모든 문자열을 리스트로 돌려줌
- `re.search()`
  - 문자열 전체를 검색하여 정규식과 매치되는지 조사함
- 
- 패턴을 명시할때 `r'문자열'` 을 붙임
  - raw string으로 백슬래시 문자를 해석하지 않고 남겨둠
  - 백슬래시 두번 사용하지 않기 위함
    - `re.compiler(r'(\d+)/(\d+)/(\d)')`
    - `re.compiler('(\\d+)/(\\d+)/(\\d+)')`

정규식 패턴 표현

- `[문자들]`
  - 문자들 중에 하나이어야 함
    - [xy] : 문자 선택을 표현하며 x와 y중 하나를 의미
  - 가능한 문자들의 집함 정의
  - ^[0-9] : 0부터 9사이 숫자중 하나로 시작하는 단어
- `[^문자들]`
  - 문자들 중 하나를 제외한 문자의미
- `[문자a-문자b]`
  - range를 뜻함
  - 문자 a ~ 문자 b 사이의 문자를 의미
- `|`
  - 두 패턴중 하나 이어야 함
- `\d`
  - 숫자 0 ~ 9
- `\w`
  - 문자를 의미
- `\s`
  - 화이트 스페이스 의미 `[\t\n\t\f]` 와 동일
- `.`
  - \n 을 제외한 모든 문자 의미
