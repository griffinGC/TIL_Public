# 연산자

## 파이썬 나눗셈

- divmod(원래 수 , 나눌 수) 하면 몫 과 나머지를 리턴
  - divmod(15, 3) = (5, 3)
- `/`
  - 나누고 소수점 이하도 가짐
- `//`
  - 나누고 소수점 이하는 버림
  - 몫을 구할때 사용
  - 혹은 `int(5 / 3)` 을 이용해도 몫을 구할 수 있음
- `%`
  - 나머지를 구할때 사용



## 파이썬 Null

- 파이썬에서 null은 없음
  - None 존재



## 콤마(`,`) 연산자

- 중첩 리스트를 만들어 주는 역할 수행

  - 대괄호를 부여한것과 동일한 역할

  ```python
  [1,[2,3]]
  ```
