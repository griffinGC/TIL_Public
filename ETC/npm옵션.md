# npm 여러가지 옵션

- `--save` 
  - `node_modules` 에 설치
  - `package.json` 파일에 `dependencies` 에 패키지 저장하는 옵션
  - `--production` 빌드시 해당 플러그인 포함됨
- `--save-dev`
  - 마찬가지로 `package.json` 파일에 저장되고 파일 내부 `devDependencies` 밑에 패키지를 저장하는 옵션
  - development 패키지 저장할때 사용
  - `--production` 빌드시 해당 플러그인 포함 안됨
